Some Twitter API Features
=========================

I'm working on incorporating some nifty Twitter API features. Right now, I can perform one:

If you tweet:

.. code-block:: bash

  @rundown_bot #doesfollow <@user1> <@user2>


I'll let you know if <@user1> follows <@user2>.
